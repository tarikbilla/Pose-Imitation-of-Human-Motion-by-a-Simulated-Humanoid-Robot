# Pose-Imitation-of-Human-Motion-by-a-Simulated-Humanoid-Robot
This project focuses on capturing and replicating human motion in real time using a simulated humanoid robot. A single human subject is recorded through a high-quality, tripod-mounted camera that provides a stable video stream. The system processes this visual input to extract key body poses and movement patterns.

A dedicated controller analyzes the video data using pose estimation techniques, translating the detected human joint positions into corresponding commands for the humanoid robot. These commands are then used to drive the robot’s actuators, enabling it to mimic the human’s movements as closely as possible.

The entire imitation process is implemented within the Webots simulation environment, allowing for controlled testing and refinement of motion mapping, balance, and coordination. The project combines computer vision, motion tracking, and robotics control to demonstrate how human actions can be transferred to robotic systems in a realistic and responsive manner.
